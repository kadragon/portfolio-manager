# Architecture Decision Records

Each ADR documents a significant architectural decision: context, options considered, and rationale.

## Format

```
docs/adr/NNNN-short-title.md
```

```markdown
# NNNN: Title

**Status:** Accepted | Superseded by [NNNN] | Deprecated
**Date:** YYYY-MM-DD

## Context
What problem or constraint drove this decision?

## Decision
What was decided?

## Consequences
What does this make easier or harder?
```
